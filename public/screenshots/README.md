# Screenshots / Captures d'écran

## EN — How to add your project screenshots

Place your screenshot files here (`public/screenshots/`).
File names must match those in `lib/data.ts` > project `images[]`:

| Project            | Expected filenames                                    |
|--------------------|-------------------------------------------------------|
| NetPass            | netpass-1.png · netpass-2.png · netpass-3.png        |
| Complaint System   | complaint-1.png · complaint-2.png                    |
| Scholarship System | scholarship-1.png · scholarship-2.png                |
| Green Hope         | green-hope-1.png · green-hope-2.png                  |

If a file is missing the modal falls back gracefully to a placeholder.

## FR — Comment ajouter vos captures d'écran

Placez vos fichiers ici (`public/screenshots/`).
Les noms doivent correspondre au champ `images[]` dans `lib/data.ts`.
Si une image est absente, la modale affiche un placeholder automatiquement.
